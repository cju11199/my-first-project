What you practice

Real registration skills, safely simulated

Every tool mirrors what you'll use on a linac — so the muscle memory transfers.

▦2D / 2D matching

Orthogonal AP + Lateral portal-to-DRR alignment with 5DOF correction (Lat / Lng / Vrt / Roll / Pitch).

◖CBCT 6DOF fusion

Three-plane volumetric registration with full 6DOF couch correction and teal/orange fusion overlay.

⊹Real 3D CT (MPR)

True multiplanar reslice through real CT volumes — axial, coronal, sagittal — with structure contours.

◉Pro tools

Color Wash, Spyglass lens, contrast windowing, blend sliders, isocenter reticle, and fusion color.

±Live error grading

Real-time residual-error readout graded against tolerance, so you know when the match is good enough.

⌨Console-like control

Drag to translate/rotate, scroll to browse slices, undo, generate fresh setup errors, keyboard shortcuts.
